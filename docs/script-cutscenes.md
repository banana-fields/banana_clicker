# Script des 8 cutscenes — Banana Clicker

**Héros :** Kiki (singe) · **Rival :** Gros Koko (gorille) · **Ton :** aventure légère  
**Humains :** en arrière-plan (usine Banana Factory) · **Fin :** liberté des singes

---

## Scène 1 — « La dernière banane du matin »
**Déblocage :** première entrée dans le jeu (après JOUER)  
**Visuel IA :** petit singe orange sous un grand bananier, brume matinale, style cartoon chaleureux

| | |
|---|---|
| **Kiki** | Encore une matinée… une banane, une seule. Les gorilles prennent tout le reste. |
| **Kiki** | Mais si je commence petit… peut-être qu’un jour les singes mangeront à leur faim. |
| **Kiki** | Allez. Une banane. Puis une autre. C’est ça, mon plan. |

---

## Scène 2 — « Premier allié »
**Déblocage :** premier Singe aide acheté  
**Visuel IA :** deux singes joyeux dans la forêt, l’un tend une banane

| | |
|---|---|
| **Bongo** | Hé ! Tu cliques comme un fou. Tu montes une révolution ou tu fais du sport ? |
| **Kiki** | Les deux. Tu veux rejoindre le clan ? |
| **Bongo** | Gratuit ? … OK. Mais je prends 10 % des bananes. |
| **Kiki** | 5 %. |
| **Bongo** | Deal. Chef. |

---

## Scène 3 — « Terres interdites »
**Déblocage :** 5 Bananiers achetés  
**Visuel IA :** plantation de bananiers, clôture au loin, silhouettes d’humains avec casques jaunes (petits, arrière-plan)

| | |
|---|---|
| **Kiki** | Cinq bananiers. On ne se cache plus. |
| **Bongo** | Au loin… l’usine Banana Factory. Les humains bossent sans voir qu’on repousse les limites. |
| **Kiki** | Ils font leur stage, on fait notre révolution. Chacun son combat. |
| **Bongo** | Les gorilles vont remarquer. |

---

## Scène 4 — « Le convoi des gorilles »
**Déblocage :** premier Camion acheté  
**Visuel IA :** camion chargé de bananes, gorilles armés de badges « Koko Corp », route forestière

| | |
|---|---|
| **Gros Koko** *(voix forte, hors champ)* | Qui ose transporter des bananes dans MON secteur ? |
| **Bongo** | C’est Gros Koko… le gorille qui pense que la gravité lui obéit. |
| **Kiki** | On ne recule pas. Le camion roule pour le clan. |
| **Gros Koko** | Tu vas regretter, petit singe. |

---

## Scène 5 — « L’usine et les humains »
**Déblocage :** première Usine achetée  
**Visuel IA :** usine banane cartoon, humains en blouse sur une passerelle (arrière-plan), singes au premier plan

| | |
|---|---|
| **Kiki** | Une usine à purée. On ne fait plus que cueillir — on transforme. |
| **Bongo** | Les humains de l’usine font « convoyeur, machine à peler »… nous on fait « liberté ». |
| **Kiki** | Ils ne savent pas qu’on existe. Mieux vaut ainsi. |
| *(bruit de moteur)* | |
| **Bongo** | Koko a doublé ses gorilles. Il a peur. |

---

## Scène 6 — « Le gorille en chef »
**Déblocage :** premier Empire Banane acheté  
**Visuel IA :** Kiki face à Gros Koko sur une colline, port et cargos en arrière-plan, tension cartoon

| | |
|---|---|
| **Gros Koko** | Tu contrôles l’économie banane ? Ridicule. Je contrôle les gorilles. |
| **Kiki** | Et bientôt, les singes contrôleront leur vie. |
| **Gros Koko** | Je vais écraser ton empire. Comme une banane trop mûre. |
| **Kiki** | On verra qui glisse en premier. |
| **Gros Koko** | … Qu’est-ce que ça veut dire ? |
| **Kiki** | Tu comprendras. |

---

## Scène 7 — « La banane du destin »
**Déblocage :** premier Temple Banane acheté  
**Visuel IA :** Gros Koko en plein élan vers Kiki, banane sur le sol, moment juste avant la chute — slapstick cartoon

| | |
|---|---|
| **Gros Koko** | PERSONNE NE PRIE LE TEMPLE BANANE ! |
| *(il charge)* | |
| **Bongo** | Koko ! Regarde tes pieds ! |
| **Gros Koko** | Quoi ? Une ban— AAAAAH ! |
| *(glissade épique sur la banane, chute)* | |
| **Bongo** | … Chef. C’était magnifique. |
| **Kiki** | La gravité est neutre. Elle choisit toujours le gorille le plus arrogant. |
| **Gros Koko** *(à terre)* | Je… je ne glisse pas. C’est… un plongeon tactique. |
| **Tous les singes** | LIBERTÉ ! |

---

## Scène 8 — « La révolution banane »
**Déblocage :** premier prestige  
**Visuel IA :** fête dans la forêt, singes libres, bananes dorées, temple au loin, humains observent depuis l’usine (arrière-plan)

| | |
|---|---|
| **Mama Banane** *(sage du temple)* | Tu as recommencé. Pas parce que tu as échoué — parce que tu es devenu une légende. |
| **Kiki** | Chaque prestige, le clan est plus fort. Koko est encore dans la boue, probablement. |
| **Bongo** | Il dit qu’il « médite sur la banane ». |
| **Kiki** | Les singes sont libres. Les humains bossent. Les gorilles… glissent. |
| **Kiki** | La révolution banane continue. |
